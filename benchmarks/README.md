# Introduction to Genai-bench

[Genai-bench](https://docs.sglang.io/genai-bench/) is a powerful benchmark tool designed for comprehensive token-level performance evaluation of large language model (LLM) serving systems.

It provides detailed insights into model serving performance, offering both a user-friendly CLI and a live UI for real-time progress monitoring.
